# Sudoku Game using Python

## Overview
This is a beginner-friendly Sudoku game built completely using Python.  
The project implements a 9x9 Sudoku board where players can fill numbers while following Sudoku rules.

The main purpose of this project is to understand:
- 2D Lists (Matrices)
- Nested Loops
- Functions
- Conditional Statements
- Game Logic
- Input Handling in Python

---

## Features
- 9x9 Sudoku Board
- Console-based Gameplay
- Row Validation
- Column Validation
- 3x3 Grid Validation
- User Input Handling
- Beginner-friendly Python Project

---

## Technologies Used
- Python

---

## Sudoku Rules
- Each row must contain numbers from 1 to 9 without repetition.
- Each column must contain numbers from 1 to 9 without repetition.
- Each 3x3 box must contain numbers from 1 to 9 without repetition.
